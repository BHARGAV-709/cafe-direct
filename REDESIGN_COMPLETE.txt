🎨 MINIMALIST DESIGN HIGHLIGHTS

What Makes It Minimalist:

1. NO GRADIENTS - All solid colors, no transitions
2. NO ROUNDED CORNERS - Sharp, clean edges (border-radius: 0)
3. NO DECORATIVE ELEMENTS - Every element has purpose
4. GENEROUS SPACING - Lots of white space
5. SIMPLE COLORS - Only 5 colors (primary, accent, secondary, bg, muted)
6. ELEGANT FONTS - Large, bold serif for headings
7. MINIMAL SHADOWS - Only essential depth
8. CLEAN LAYOUT - Clear visual hierarchy
9. QUALITY IMAGERY - Professional, high-quality photos
10. FOCUS ON CONTENT - Nothing distracts from the message

🚀 HOW TO GET STARTED

STEP 1: View the Website
────────────────────────────────────────────────────────────────────────────
Run the development server:
  $ cd /vercel/share/v0-project
  $ pnpm dev

Open in browser:
  http://localhost:3000

STEP 2: Update Your Content
────────────────────────────────────────────────────────────────────────────
Edit these files:
  • app/page.tsx → Homepage text
  • app/menu/page.tsx → Menu page text
  • lib/menu-data.ts → Menu items and prices

STEP 3: Replace Images
────────────────────────────────────────────────────────────────────────────
Upload your photos to:
  • /public/hero-cafe.jpg → Hero image
  • /public/pastries.jpg → Menu preview
  • /public/food/*.jpg → Menu items

STEP 4: Update Business Info
────────────────────────────────────────────────────────────────────────────
Change in app/page.tsx footer:
  • Hours of operation
  • Location address
  • Phone number
  • Email address

STEP 5: Set WhatsApp Number
────────────────────────────────────────────────────────────────────────────
Update in app/menu/page.tsx:
  const whatsappNumber = 'YOUR_NUMBER_HERE';

STEP 6: Deploy
────────────────────────────────────────────────────────────────────────────
Push to GitHub and deploy to Vercel:
  $ git push origin main

❓ COMMON QUESTIONS

Q: Can I change the colors?
A: Yes! Update app/globals.css --color definitions

Q: Can I add more menu items?
A: Yes! Edit lib/menu-data.ts and add new items

Q: Can I use different fonts?
A: You can modify app/layout.tsx but keep 2 fonts maximum

Q: Can I change the layout?
A: Yes, but follow the minimalist principles for consistency

Q: Can I add animations?
A: Minimal animations only - keep them subtle and purposeful

Q: Can I add more sections?
A: Yes, follow the section spacing pattern (80px-128px)

Q: How do I add user accounts?
A: Integrate with Supabase or another auth provider

Q: How do I track orders?
A: Connect to a backend database and admin dashboard
