✨ MAIN FUNCTIONS
─────────────────────────────────────────────────────────────────────────────
Function              What It Does            How to Use
─────────────────────────────────────────────────────────────────────────────
Add to Cart           Adds item, updates      Click item card button
                      cart count
Category Filter       Switch menu view        Click category button
                      (Starters, Main, Drinks)
Quantity +            Increases qty           Click + button in cart
Quantity -            Decreases qty           Click - button in cart
Remove Item           Deletes from cart       Click X button in cart
Checkout              Opens WhatsApp with     Click "Order via WhatsApp"
                      pre-filled order
Form Validation       Checks required         Validates before checkout
                      fields
WhatsApp Order        Sends to café           Pre-fills all details
QR Code System        Table tracking          Auto-extracts ?table=X
Dark Mode             Switches theme          System preference
